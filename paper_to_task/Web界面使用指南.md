# 🌐 Paper-to-Task Web界面使用指南

## ✅ Web应用已启动！

**访问地址：**
- 本地访问：http://localhost:5000
- 局域网访问：http://127.0.0.1:5000

---

## 🎯 体验流程

### 第一步：打开Web界面
在浏览器中打开上面的访问地址，您将看到一个漂亮的界面：
- 渐变色背景
- 文件上传区域
- 实时处理进度

### 第二步：上传PDF文件
1. 点击上传区域或拖拽PDF文件
2. 支持16MB以内的PDF文件
3. 系统会显示文件名

### 第三步：开始处理
1. 点击"开始处理"按钮
2. 观察实时进度条
3. 等待处理完成（约15-20秒）

### 第四步：查看生成结果
处理完成后会自动显示：

**📊 质量评分卡片**
- 总分：0.73/1.00
- 等级：A/B/C/D
- 四维度评分：完整性、准确性、清晰度、可行性

**📋 任务描述**
- 显示生成的任务描述
- 数据文件数量

**📊 评分项**
- 显示所有评分项
- 每项的权重和关键词

### 第五步：提供反馈改进（重点功能！）
1. 在"💡 提供反馈改进"区域输入您的建议
2. 例如：`checklist的评分项需要更具体，要包含具体的模型名称和准确率指标`
3. 点击"提交反馈并改进"
4. 观察质量评分提升！

### 第六步：创建项目
1. 在"🚀 创建项目"区域输入项目名称（可选）
2. 点击"创建sci_tasks项目"
3. 查看生成的项目路径
