Run 1 — Full ReproLR Pipeline Implementation
--------------------------------------------------------------------------------

What was implemented:
  A complete four-stage OLS regression pipeline in code/experiment.py:

  1. Data Ingestion  — Load synthetic_data.csv (1000 samples, 5 features),
                       split 80/20 train/test.
  2. Preprocessing   — Feature standardization (Z-score via training mean/std),
                       response centering, condition-number check via SVD.
  3. Model Fitting   — Pivoted QR decomposition (scipy.linalg.qr with
                       pivoting) → triangular solve → coefficient recovery
                       on original feature scale.
  4. Evaluation      — MSE, RMSE, R² computation; report generation with
                       scatter plot and residual histogram saved to report/.

  The implementation follows the algorithm specification in INSTRUCTIONS.md,
  including the two-stage coefficient recovery (center response → solve on
  standardized design → back-transform to original scale).
