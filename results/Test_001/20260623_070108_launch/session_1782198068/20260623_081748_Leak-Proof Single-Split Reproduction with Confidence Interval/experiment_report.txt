Run 1 (Initial Implementation)
================================================================================

What was implemented/modified:
- Implemented the full Leak-Proof Single-Split protocol from scratch.
- Key steps implemented:
  1. Data loading (1000 samples, 5 features x1-x5, target y)
  2. Deterministic 80/20 train-test split (random seed = 42)
  3. Leak-proof Z-score standardization (mean & std computed on training set
     only, then applied to both train and test sets)
  4. OLS linear regression training via sklearn LinearRegression
  5. Test-set evaluation (MSE, RMSE, R², MAE)
  6. 95% t-confidence interval for the true expected MSE (t_crit ≈ 1.972,
     df = 199)
  7. 5 visualization figures generated (predicted vs actual, residual analysis,
     squared errors with CI, model coefficients, predictions vs actuals
